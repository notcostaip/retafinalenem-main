<!-- HOW TO RUN -->
<div align="center">

### &nbsp; 🚀 Como Executar Localmente

</div>

### 1. Clone o repositório
```bash
git clone https://github.com/notcostaip/retafinalenem-main.git
cd retafinalenem-main
```

### 2. Instale as dependências
```bash
npm install
```

### 3. Inicie o servidor
```bash
npm run dev
```

### 4. Build para Produção
```bash
npm run build
```
